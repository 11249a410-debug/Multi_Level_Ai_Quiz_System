# Multi_Level_Ai_Quiz_System
Multi Level AI Quiz System is an interactive web app with 15 levels covering AI, ML, IoT, GenAI, and Agentic AI. It evaluates users by score and time, shows top 25 leaderboards, and includes mistake review for better learning.
# 🚀 Multi Level AI Quiz System

An interactive web-based quiz platform with 15 levels covering:
- Artificial Intelligence
- Machine Learning
- IoT
- Generative AI
- Agentic AI

## ✨ Features
- 🎯 15 progressive levels
- 🏆 Top 25 leaderboard per level
- ⏱️ Score + time-based ranking
- ❌ Mistake review system
- 📊 Progress tracking
- 🎨 Modern UI

## 🚀 How to Run
1. Download or clone the repo
2. Open `index.html` in browser
